# 🪨📄✂️ Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML and JavaScript**.
Play against the computer and see if you can win! 🎮

---

## 🚀 Features

* 🎲 Random computer move generation
* 🧠 Game logic for win / lose / tie
* ⚡ Instant result using alerts
* 🖱️ Easy-to-use buttons (Rock, Paper, Scissors)

---

## 🛠️ Technologies Used

* **HTML5**
* **JavaScript (Vanilla JS)**

---

## 📂 Project Structure

```plaintext
rock-paper-scissors/
└── index.html
```

---

## 🎮 How to Play

1. Open the `index.html` file in your browser
2. Click on any option:

   * 🪨 Rock
   * 📄 Paper
   * ✂️ Scissors
3. The computer will randomly choose its move
4. An alert will show:

   * Your choice
   * Computer's choice
   * Result (Win / Lose / Tie)

---

## 🧠 Game Logic

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* Same choices = Tie
